PROJECTS

The Employee Management System for a Vehicle Service Station

MERN stack project that enables efficient management of employees. My part of the project offers CRUD operations, auto-generates QR codes for Employee, Read QR code and get attendance, salary calculator to calculate employee salaries, generates monthly salary report, and auto-generate Employee ID cards.

  • Technologies-: Mongo DB, React JS, Node JS, Express, Bootstrap, Redux, jspdf, qrcode.react

Online voting system for Awards Nomination

Web application developed for award nominations and voting online. Users can effortlessly browse categories, choose nominees, and cast their votes. Additionally, the system provides administrators with access to view the voting count, ensuring transparency and oversight.

  • Technologies-: HTML, CSS, JavaScript, PHP, MYSQL

Online Channel booking system for Hospital

Web-based application where users can access the system online and schedule appointments with doctors of their choice. This system eliminates the need for in-person bookings and allows users to select their preferred doctor and desired appointment time.

  • Technologies-: HTML, JavaScript, PHP, MYSQL, Bootstrap

MegaMarket App for local self-entrepreneurs

“MegaMarket” is a mobile application that made for local self-entrepreneurs as solution for the economic crisis. The purpose of this mobile application project is to encourage and increase the productivity of local self-employed entrepreneurs in Sri Lanka, while also contributing to the country's economic growth. The project aims to offer a platform for local entrepreneurs to promote their products and services, not only in the local market but also in international markets, thereby increasing their revenue potential.

  • Technologies-: Kotlin, Firebase

Online banking system

The Online Banking System is a web application that offers convenient access to bank accounts. Users can securely log in, view balances, transactions, transfer funds, pay bills, add cards, and apply for loans.

  • Technologies-: SP & Servlets, Java, MYSQL Workbench, Bootstrap.

School-management-system

This project was created using the MERN stack, with function components and includes full CRUD operations as well as search functionality. The project was created for self-study purposes

  • Technologies-: Mongo DB, React JS, Node JS, Express, Bootstrap, Redux

Artificial Intelligence Assistant Athena
